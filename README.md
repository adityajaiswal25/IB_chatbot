# IB Group Chatbot 🌾

**Live Demo:** https://adityajaiswal25.github.io/IB_chatbot/

## Project Description

IB Chatbot is a mobile-optimized, fullscreen AI chatbot for **IB Group** — India's leading integrated protein company specializing in poultry, livestock feeds, and farmer empowerment programs.

The chatbot provides instant information about:
- 🐔 Poultry feeds and nutrition (ABIS Gold & ABIS Broiler series)
- 🏠 Parivartan EC House upgrade program
- 💉 Hatchery vaccination services (In-ovo Innovo® technology)
- 🔬 APRDC diagnostics and disease management
- 📱 Broiler Technology Platform (BTP)
- 👨‍🌾 Farmer support and integration programs

---

## 🎯 Features

- **Mobile-First Design** — Fully responsive, fullscreen interface optimized for all devices
- **Real-time Chat** — Instant AI responses with typing animations
- **Demo Mode** — Pre-built responses covering IB Group products and services
- **RAG-Ready** — Integrated with n8n RAG system using comprehensive knowledgebase
- **Quick Reply Buttons** — One-tap access to popular queries
- **Safe Area Support** — Works seamlessly on notch phones and landscape mode
- **Lightweight** — Single-file HTML implementation (19KB)
- **Professional UI** — IB Group branded colors (green & gold) with smooth animations

## 🚀 Usage

### Local Testing
1. Download or clone the repository
2. Open `index.html` directly in any browser
3. Chat with the bot using demo responses

### Deployment with GitHub Pages
The project is already hosted on GitHub Pages:
- **Live URL:** https://adityajaiswal25.github.io/IB_chatbot/
- Any push to the `main` branch automatically updates the live site

### n8n RAG Integration
To enable AI-powered responses:
1. Replace `N8N_WEBHOOK_URL` in `index.html` with your actual n8n webhook
2. The chatbot will send user queries to your RAG pipeline
3. Responses will be generated from the knowledgebase

**Current n8n Webhook:** Already configured (can be updated in line 407 of index.html)
