# IB Group Chatbot

## Project Description

IB Chatbot is a mobile-optimized fullscreen chatbot for IB Group, India's leading integrated protein company specializing in poultry, livestock feeds, and farmer programs.

## Files
- `index.html` - Complete single-file chatbot (HTML/CSS/JS)
- `knowledgebase.md` - IB Group knowledgebase with poultry feeds, Parivartan details, APRDC diagnostics, and contacts

## Features
- Mobile-first fullscreen interface
- Demo responses covering IB Group products and services
- Typing animations and quick replies
- n8n RAG integration ready (uses knowledgebase.md)

## Usage
Open `index.html` in any browser. Demo includes info on ABIS feeds, broiler integration, and farmer support.

🌾 For IB Group farmers and stakeholders.
